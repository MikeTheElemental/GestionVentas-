Historia de Usuario 1

Título:
Registrar una venta.

Descripción:
Como cajero quiero registrar una venta para almacenar la información de los productos vendidos.

Objetivo:
Permitir registrar correctamente una venta.

Criterios de aceptación:
- Debe permitir agregar productos.
- Debe calcular el subtotal.
- Debe calcular el total.
- Debe generar la factura.
